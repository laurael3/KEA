Tema 2

Grundlæggende Web

Om temaet

I Tema 1 blev vi introduceret til hvad uddannelsen handlede om og hvad vi kunne forvente at lære. Derefter begyndte vi at lære noget html, med fokus på semantisk markup som skulle ligge en grundsten for vores opgave i tema 2.

Vi lavede først en hjemmeside med frie tøjler, lavet med Mobile-first i mente. Derefter fik vi udleveret nogle Wireframes og Layoutdiagrammer som vi skulle replikere på vores Website version af vores hjemmeside. Her lærte vi lidt om Css grids gennem nogle sandbox-øvelser, og vi fik lavet nogle media queries så siden endte med at blive responsiv.

Så til sidst havde jeg en mobil og desktop version af min aller første hjemmeside på mit eget domæne. Og jeg havde fået lidt forståelse for html-sproget, og lært nogle forskellige ting man kunne med Css som f.eks grids.
